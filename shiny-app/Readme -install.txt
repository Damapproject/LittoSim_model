- Installer R & Rstudio 
- Lancer Rstudio et depuis Tools\Download Package, installer le package "Shiny"
- Dans Rstudio copier coller la ligne suivante
install.packages(c("ggplot2","stringr","dplyr","plyr"), dependencies = T)
Et cliquer Entrer pour charger d'autres packages
- Télécharger depuis le GitHub LIttoSIM l'appli développé pour LittoSIM appelé Shiny-app
- Dans RStudio faite un OpenFile de cette appli
- Puis cliquer sur Run APP
